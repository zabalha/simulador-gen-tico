#  Simulador Genético Familiar

Aplicação desktop desenvolvida em Python com interface gráfica (Tkinter) para simulação educacional de herança genética familiar.

---

##  Sobre o Projeto

O Simulador Genético Familiar permite que o usuário insira características físicas e histórico de doenças de:

- Pai
- Mãe
- Avós paternos
- Avós maternos

Com base nos dados inseridos, o sistema estima:

- Probabilidade de características físicas
- Altura estimada
- Risco estatístico de doenças hereditárias

 Modelo educacional simplificado (não substitui avaliação médica).

---

##  Conceitos Aplicados

- Herança Mendeliana simplificada
- Modelo poligênico simplificado
- Estatística básica aplicada à agregação familiar
- Interface gráfica com Tkinter
- Organização modular de funções
- Geração de executáveis multiplataforma

---

##  Tecnologias Utilizadas

- Python 3
- Tkinter
- PyInstaller
